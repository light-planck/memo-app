# memo-app

## 起動方法

### git clone

```bash
git clone https://github.com/light-planck/memo-app.git

cd memo_app

git checkout development
```

### セットアップ

```bash
./setup.sh
```

### 起動

```bash
bundle exec ruby app.rb
```

http://localhost:4567 にアクセス

## ERB Lint

```bash
bundle exec erblint --lint-all
```
